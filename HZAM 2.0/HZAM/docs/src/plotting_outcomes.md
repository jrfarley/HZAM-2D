# Plotting outcomes

The functions for plotting the outcome of a single simulation are [`HZAM.plot_fitnesses`](@ref), [`HZAM.plot_population_tracking_data`](@ref), and [`HZAM.plot_phenotypes`](@ref).

The functions for plotting the outcome of a set of simulations are [`HZAM.plot_output_field`](@ref) and [`HZAM.summarize_gene_correlations`](@ref).
